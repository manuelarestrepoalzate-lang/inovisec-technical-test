# Inovisec Technical Test

## Descripción

Aplicación desarrollada en React para visualizar casos georreferenciados mediante un mapa interactivo.

## Funcionalidades

- Login
- Dashboard
- Visualización de casos en mapa
- Selección dinámica de casos
- Panel lateral con detalle del caso

## Tecnologías

- React
- Vite
- JavaScript
- Leaflet
- React Leaflet

## Instalación

```bash
npm install
npm run dev
```

## Estructura

src/
├── components/
│ └── MapView.jsx
├── data/
│ └── cases.js
